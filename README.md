# The Unsafe Engine

Welcome to the official repository for the Unsafe Engine. Published under [Unsafe Games](https://unsafe.games), this engine is a cinematic, player-facing, single-roll, rules-lite, OSR-style, 3d6 roll-under tabletop RPG framework designed for high-stakes survival, granular resolution, and a progression to 12th-level. This system is designed for group, GM-less, and solo gameplay, with a built-in oracle-style resolution system.

This repository serves as the central hub for the development of the system reference documentation (SRD), tracking the core systems, character options, and accessories.

More summerized information will be posted to this readme as development progresses.
